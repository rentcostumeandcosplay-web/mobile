## 🚀 Hướng dẫn cài đặt và chạy project

### Cài đặt các thư viện phụ thuộc (Dependencies)
Chạy lệnh sau trong thư mục gốc của project:
```bash
npm i
```

Chạy lệnh sau để khởi động project:
```bash
npx expo start
```

Về phía pc thì phải cài python và cài các thư viện rồi chạy script.py để có thể mở kết nối cho điện thoại
